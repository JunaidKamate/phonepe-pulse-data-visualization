# PhonePe Pulse Data Visualization & Exploration

## Overview
Interactive Power BI dashboard analyzing PhonePe Pulse data (2018–2023) to explore digital payment trends across India. The project includes geographic insights, transaction patterns, user growth, and category-level analytics using fully interactive slicers and KPIs.

## Dataset
Processed CSV files derived from the official PhonePe Pulse GitHub repository:
- aggregated_transaction
- aggregated_user
- map_transaction
- map_user
- top_transaction
- top_user

## Tools
Power BI Desktop, Power Query, DAX, GitHub

## Features
- State-level transaction map
- Quarterly transaction trend
- Transaction type comparison
- Top states and districts
- User growth trend
- Gauge visual for Average Transaction Value (ATV)
- KPI cards for key metrics
- Ten interactive dropdown slicers for detailed filtering

## Key Insights
- Strong growth in total transaction value, exceeding ₹121T.
- Telangana, Maharashtra, and Karnataka lead in digital payments.
- Bengaluru, Hyderabad, and Pune show highest district-level activity.
- Peer-to-peer transfers dominate overall transaction value.
- Average Transaction Value is approximately ₹1.69K.
- Registered users continue to grow, crossing 300M+.
- Tier-2 and Tier-3 regions show increasing adoption.
